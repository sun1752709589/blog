# 关于写作这件事
```ruby
今天来聊聊关于写作的事。

首先，对于完全无写作欲望的人，你可以不用往下看了，因为看了也是浪费你的时间。

聊到写文章这事呢，很多人其实有一个误区就是：等我准备好以后，我就咋咋咋。。。
大部分人迟迟不能开始的一个重要原因就是：想要一次就做出很牛的作品。
其实笔者我之前也有这个顾虑吧，总怕写的太烂被喷，这个确实挺伤自尊的。不过有时候学学鸵鸟感觉也还不错哈。

其次要指出的是，最有效的学习方法就是把自己知道的教给别人，而写作可以做到不依靠你自己就可以独立完成教别人的过程。
想想一件事情你只需要做一遍然后其他n个人看到并且学习了，这个过程完全不需要你再干预。是不是很爽？起码成就感爆棚吧。

再次至于文笔，即所谓的文采，这东西重要，但是和你写的东西影响了n个人相比，是次要的。

最后，写作之后你可以得到别人的真实反馈。和比人交流想法，更新自己的操作系统(思想)。

写吧，只要你用心去写，那你一定是在用心思考！写你想写的任何东西！

```
